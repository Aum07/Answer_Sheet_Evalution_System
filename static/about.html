About Our Project

The Answer Sheet Evaluation System is a final year project developed at D Y Patil International University that uses advanced Natural Language Processing and Machine Learning techniques to automate the evaluation of written answers.

Project Overview

This system was developed to address the time-consuming and subjective nature of manual answer sheet evaluation. By leveraging state-of-the-art NLP techniques, our system can:

  • Process multiple answer sheets simultaneously
  • Compare student answers with reference solutions
  • Evaluate based on semantic similarity, not just keyword matching
  • Consider factors like readability, coherence, and structure
  • Generate detailed feedback for each answer
  • Provide consistent scoring across all evaluations

How The System Works

1
PDF Upload & Text Extraction

The system accepts reference and student answer PDFs, extracting text using OCR when needed.

2
Question-Answer Parsing

Text is parsed to identify questions and corresponding answers from both reference and student sheets.

3
Feature Extraction

The system extracts semantic similarity, part-of-speech features, readability metrics, and more.

4
Score Prediction

A trained machine learning model predicts scores based on extracted features.

5
Feedback Generation

The system generates personalized feedback based on the evaluation results.
