Frequently Asked Questions

Our system achieves high accuracy by using advanced NLP techniques and machine learning models. In our testing, it has shown a correlation of over 85% with human evaluators. The system considers semantic similarity, not just keywords, making it more effective than traditional automated systems.

Currently, the system supports PDF files. We recommend using digital PDFs for best results, but the system can also process scanned PDFs using OCR technology. Each PDF should contain clearly marked questions and answers.

The system can process multiple answer sheets simultaneously. For optimal performance, we recommend uploading up to 50 student answer sheets at a time, though the system can handle more depending on server capacity.

The system works best for subjects where answers can be evaluated based on textual content, such as humanities, social sciences, and theoretical components of technical subjects. It may not be suitable for evaluating mathematical equations, diagrams, or code snippets without additional customization.
